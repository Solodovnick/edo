# Контекст канвас-сессии — EDO Bank
**Версия:** 1.0.2
**Дата:** 04.05.2026

## Связанные файлы
- `docs/canvas/edo-canvas.drawio` — Draw.io диаграмма
- `docs/canvas/index.html` — интерактивный HTML-канвас
- `CONTEXT-AND-SUMMARY.md` (корень) — полный саммари сессии

## Что содержит канвас
Визуальное представление архитектуры проекта EDO Bank:
- Фронтенд: React 18 + TypeScript + Vite + Tailwind + shadcn/ui
- Бэкенд: Spring Boot 3.2.5 + PostgreSQL 15 (Docker, порт 5433)
- Документация: FR, ER-диаграмма, State-диаграмма, ADR, Глоссарий

## Контекст проекта
Модуль ЭДО (Электронный Документооборот) для EDO Bank.
Цель: повышение KPI обработки устных обращений с 91% до 99%.
Горизонт: 6 месяцев (MVP) + 3 месяца (расширение).
