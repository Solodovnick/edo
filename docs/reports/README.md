**WakaTime:** расширение [vscode-wakatime](https://marketplace.visualstudio.com/items?itemName=WakaTime.vscode-wakatime) шлёт время кодирования на [wakatime.com](https://wakatime.com) — графики открываются **в личном кабинете** на сайте, а не внутри этого HTML. В отчёте — ссылка и плейсхолдер для скриншота/вставки публичного embed, если включите share в настройках WakaTime.

Открытие HTML: из корня репозитория двойной клик по файлу или в VS Code / Cursor — **ПКМ → Open with Live Server** (если установлен) или открыть в браузере.
