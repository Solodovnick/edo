# UI-артефакты (исторические заметки по интерфейсу)

Здесь собраны markdown-файлы с описаниями экранов, кабинетов, доработок UX/UI и смежных технических заметок, ранее лежавшие в корне репозитория.

**Активные требования и спеки продукта:** каталог [`docs/`](../) — `functional-requirements.md`, `ui-ux-brief.md`, `design-system-plan.md`.

## Подпапки

| Путь | Содержание |
|---|---|
| [`reference/`](reference/) | Архивный CSS темы shadcn (`default_shadcn_theme.css`). Рабочие токены приложения подключаются через **`src/styles/theme.css`**. |
