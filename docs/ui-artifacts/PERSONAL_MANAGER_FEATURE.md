# ✅ Новая функция: "Есть персональный менеджер"

## 📋 Что добавлено:

### 1. Новый атрибут `hasPersonalManager`
- **Тип**: boolean
- **По умолчанию**: false
- **Назначение**: Управление наличием персонального менеджера у клиента

### 2. UI компонент - Toggle кнопки "Есть" / "Нет"

**Расположение**: После блока "ФИО Регистратора"

**Функционал:**
- ✅ Две кнопки для выбора: "Есть" / "Нет"
- ✅ Claymorphism стиль с градиентом для активной кнопки
- ✅ При выборе "Нет" - скрывается поле ввода ФИО менеджера и сбрасывается VIP статус
- ✅ При выборе "Есть" - показывается поле для ввода ФИО и чекбокс VIP

### 3. Условное отображение полей

**Если выбрано "Есть":**
