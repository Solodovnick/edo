# Responsible Cabinet Redesign - Кабинет ответственного

## ✅ Реализовано: 01.03.2026

### Цели доработки:

1. **Создать "Кабинет ответственного"** - таблица обращений как экран по умолчанию
2. **При клике на обращение** - открывать карточку решения обращения
3. **Использовать дизайн-систему** карточки решения для кабинета ответственного (ретро-футуристический стиль Winamp)

## 🔄 Логика фильтрации

### Filter Logic:

```typescript
const filteredAppeals = mockAppeals.filter(appeal => {
  // 1. Фильтр по типу
  let typeMatch = true;
  if (activeFilter === 'Физ лицо') typeMatch = appeal.type === 'Физ лицо';
  if (activeFilter === 'Юр лицо') typeMatch = appeal.type === 'Юр лицо';
  if (activeFilter === 'Регулятор') typeMatch = appeal.type === 'Регулятор';
  // 'Все обращения' и 'Мои обращения' показывают всё
  
  // 2. Фильтр по поиску
  const searchMatch = 
    searchQuery === '' || 
    appeal.id.includes(searchQuery) ||
    appeal.applicantName.toLowerCase().includes(searchQuery.toLowerCase()) ||
    appeal.organizationName.toLowerCase().includes(searchQuery.toLowerCase()) ||
    appeal.responsible.toLowerCase().includes(searchQuery.toLowerCase());

  return typeMatch && searchMatch;
});
```

### Примеры фильтрации:

| Фильтр | Результат |
|--------|-----------|
| **"Мои обращения"** (по умолчанию) | Показывает все 3 обращения |
| **"Физ лицо"** | Показывает только обращение #123123444 (Дудченко Л.В.) |
| **"Юр лицо"** | Показывает только обращение #123123223 (Рога и копыта) |
| **"Регулятор"** | Показывает только обращение #123134444 (Центробанк) |
| **Поиск "Солодовников"** | Показывает 2 обращения (123123444 и 123134444) |
| **Поиск "Рога"** | Показывает 1 обращение (123123223) |
