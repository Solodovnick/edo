**Предусловия:**

Пользователь имеет права на управление шаблонами.

**Триггер:** Пользователь дает команду: Создать шаблон ответа

**Постусловие:**

Системой создан Шаблон ответа и добавлен  в список доступных шаблонов для использования.



**Основной поток:**

1. Пользователь дает команду создать Шаблон ответа;

2. Система отображает форму  Шаблон ответа с полями:



    - Наименование шаблона

    - Описание шаблона;

    - Структура шаблона;

1. Пользователь заполняет поля.

2. Пользователь дает команду Сохранить шаблон ответа

3. Система убеждается что обязательные поля заполнены.

4. Система убеждается в отсутствии дубликата [Название] в базе шаблонов.

5. Система сохраняет Шаблон ответа в БД.

6. Сценарий окончен.

**Расширения:** 

5а  Не заполнено обязательное поле Название

5а.1 Система убеждается, что поле Название - не заполнено.

 5а.2 Система сообщает пользователю «Укажите название Шаблона ответа».

5а.3 Сценарий возвращается на шаг 2 Основного потока.



5b  Не заполнено обязательное поле  Описание шаблона

5b.1 Система убеждается, что поле  Описание шаблона - не заполнено.

 5b.2 Система сообщает пользователю «Введите описание шаблона».

5b.3 Сценарий возвращается на шаг 2 Основного потока.



7a Существует дубликат шаблона 

7а.2 Система сообщает пользователю "Такое название Шаблона ответа уже существует".

7а.3 Сценарий возвращается на шаг 4 Основного потока.
