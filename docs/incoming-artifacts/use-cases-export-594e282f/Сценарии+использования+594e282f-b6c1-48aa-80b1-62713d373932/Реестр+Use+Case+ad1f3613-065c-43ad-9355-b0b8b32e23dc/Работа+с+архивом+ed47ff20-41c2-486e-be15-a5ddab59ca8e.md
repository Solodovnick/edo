|**Действующее лицо:** Сотрудник ДУКО с правами обработки обращения, Сотрудник отделения Банка, Аудитор, Руководитель, Система|
|**Предусловия:** 
У обращения статус сменился на Пройден аудит (AU-03)
Система переместила обращение в архив (AU-03)
**Триггер:** Пользователь дает команду Открыть обращение
**Постусловие:** Обращение доступно для просмотра|
|**Основной поток:** 
1. Пользователь открывает реестр обращений и выбирает фильтр "В архиве"
2. Система отображает все обращения со статусом "В архиве"
3. Пользователь открывает обращение из списка
4. Система отображает обращение
5. В после обращения доступны все атрибуты и история обращения
6. Конец сценария
