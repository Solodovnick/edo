|**ID:** Au.02|
|-|
|**Название:** Сформировать задачу для автодозвона роботом|
|**Действующее лицо:** Ответственный Аудитор, Система, Бот IP телефонии, Заявитель|
|**Предусловия:** 
Если хотя бы одно обращение в статусе "Решено"
Обращение было распределено автоматически из пула ответственному аудитору со статусом "Аудит" (Au-xxxx)
**Триггер:** Пользователь нажимает на кнопку "Выполнить аудит"
**Постусловие:** Открыта карточка Обращения|
|**Основной поток:** 
1. Пользователь делает команду "Создать автодозвон"
2. Система запрашивает подтверждение
3. Пользователь подтверждает команду
4. Система формирует задачу Боту IP телефонии на обзвон Заявителя
5. Бот IP телефонии выполняет звонок согласно заданного алгоритма (обзвон раз в сутки, шаг 1 час, исключать выходные, исключать выходные, максимум циклов - 3)
6. Заявитель принимает звонок
7. Бот IP телефонии собирает обратную связь у Заявителя
8. Бот IP телефонии заканчивает звонок
9. Бот IP телефонии сохраняет аудиосообщение с обратной связью
10. Бот IP телефонии выполняет транскрибацию аудиозаписи
11. Система подгружает файлы аудиосообщения и транскрибации в поле "Вложения"
12. Пользователю доступны для просмотра и прослушивния файлы в поле "Вложения"
13. Сценарий окончен|
|**Расширение**:
6а1. Заявитель не принимает/сбрасывает звонок Бота IP телефонии.
6а2. Бот IP телефонии выполняет автодозвон согласно алгоритма (звонить раз в сутки, шаг 1 час, исключать выходные, максимум циклов - 2)
6а3. Бот IP телефонии выставляет в поле "Время сбора обратной связи" время последнего обзвона
6а4. Бот IP телефонии отправляет уведомление ответственному Аудитору
6а5. Сценарий окончен|
