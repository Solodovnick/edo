|**ID:** Au.01|
|-|
|**Название:** Работа с уведомлением|
|**Действующее лицо:** Ответственный Аудитор, Система|
|**Предусловия:** 
Если хотя бы одно обращение в статусе "Решено"
Обращение было распределено автоматически из пула ответственному аудитору со статусом "Аудит" (Au-xxxx)
**Триггер:** Пользователь нажимает на уведомление
**Постусловие:** Открыта карточка Обращения|
|**Основной поток:** 
1. Система отправляет уведомление пользователю (Ответственному аудитору) с задачей на аудит и ссылкой на обращение.
2. Пользователь открывает уведомление (например, «Поступила задача номер *****») и нажимает на него.
3. Система открывает карточку обращения, связанную с задачей.
4. Сценарий окончен|
|**Расширение**:
1a. Уведомление недействительно или устарело
1a1. Пользователь нажимает на уведомление, но система не находит связанное обращение или задачу.
1a2. Система отображает сообщение «Связанное обращение не найдено» и не открывает карточку.
1a3. Сценарий окончен.
