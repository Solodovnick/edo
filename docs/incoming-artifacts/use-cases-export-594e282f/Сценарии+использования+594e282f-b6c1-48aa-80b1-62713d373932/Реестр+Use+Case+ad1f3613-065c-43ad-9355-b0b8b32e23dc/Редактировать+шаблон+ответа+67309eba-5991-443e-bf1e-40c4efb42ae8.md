**Предусловия:**

Пользователь имеет права на редактирование  шаблонов ответа.

Система отображает список Шаблонов Ответа 
**Постусловие**

 Шаблон ответа  обновлен и добавлен  в список доступных шаблонов для использования.



**Основной поток:**

1. Пользователь дает команду открыть Шаблон ответа;

2. Система отображает форму  Шаблон ответа с полями: 

- Наименование Шаблона ответа;

- Описание Шаблона;

- Структура Шаблона ответа

- Активность.

1. Пользователь дает команду Редактировать Шаблон ответа.

2. Система дает возможность редактировать поля.

3. Пользователь дает команду Сохранить Шаблон ответа.

4. Система убеждается что обязательные поля заполнены.

5. Система убеждается в отсутствии дубликата Наименование Шаблона ответа в базе шаблонов.

6. Система сохраняет Шаблон ответа в БД.

7. Сценарий окончен.



**Расширения:** 

5а Пользователь дает команду Сохранить_как 

5а.1 Система  дает возможность пользователю редактировать поле Наименование Шаблона ответа.

5а.2 Сценарий возвращается на шаг 6 Основного потока.



8а Существует дубликат шаблона 

8а.1 Система сообщает пользователю "Такое наименование Шаблона ответа уже существует".

8а.3 Сценарий возвращается на шаг 5 Основного потока.



8b Система НЕ сохраняет  изменения.

8b.1 Система сообщает пользователю, что не удалось сохранить изменения.

8b.1Сценарий окончен.
