**Предусловия:**

Обращение зарегистрировано в системе



**Основной поток:**

1. Система отображает страницу Кабинет руководителя с пулом обращений.

2. Система дает возможность отрыть форму Карточка обращения.

3. Пользователь открывает  Карточку обращения.

4. Система отображает форму Карточка обращения со следующими полями:

    -  заявитель (ФИО, номер телефона, тип заявителя, адрес)

    - данные обращения( номер обращения, дата регистрации, тип, категория, срок исполнения,  статус)

    - Содержание обращения

    - Решение обращения

    - Ответ клиенту

    - Вложения

    - запросы в смежные подразделения

    - ответы подразделений.

1. Руководитель дает команду "Закрыть" на странице Карточка обращения.

2. Система возвращает Руководителя на страницу "Кабинет руководителя".

3. Сценарий завершен.



**Расширение:**

 4а Ошибка доступа к базе данных.

4а1. Система уведомляет пользователя о том, что страница Карточка обращения не доступна к просмотру.

4а2. Сценарий завершен.



**Постусловия:**

Руководитель ознакомлен с содержимым обращения 

Событие 'Просмотр' зафиксировано в истории обращения (время, ID пользователя)
