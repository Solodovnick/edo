**Шаблон сценария использования включает в себя следующие элементы:**

|Термин|Определение|
|-|-|
|Предусловие|Начальные условия, при которых сценарий может быть выполнен. Они определяют, в каком состоянии должна находиться система перед запуском сценария.|
|Триггер|Событие, инициирующее выполнение сценария.|
|Постусловие|Результат выполнения сценария, который не обязательно выражается в явной форме (например, изменение состояния системы, появление новых данных или завершение операции). В постусловии могут быть указаны дополнительные изменения в системе, которые не явны исходя из названия сценария, но важны для корректной работы системы и других процессов.|
|Основной поток|Последовательность действий, приводящая к успешному завершению процесса.|
|Альтернативные потоки|Ответвления от основного сценария, возникающие при различных условиях (например, ошибка ввода, отсутствие доступа).|
