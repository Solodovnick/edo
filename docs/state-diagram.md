## Черновик состояний (из FR-02.1)

```
[Новое]
    │ Ответственный берёт в работу
    ▼
[В работе]
    │ Запрос к смежному подразделению
    ▼
[Запрос в БЛ] ────► [В работе] (получен ответ)
    │
    │ Решение подготовлено
    ▼
[Решено]
    │ Аудитор проверяет
    ▼
[Проверено аудитором]
    │ Ответ направлен заявителю
    ▼
[Закрыто]
    │ По истечении срока хранения
    ▼
[Архив]
```
