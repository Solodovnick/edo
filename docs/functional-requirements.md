## FR-01 Регистрация обращений

*Приоритет: P0*

| ID | Требование | Приоритет |
|---|---|---|
| FR-01.1 | Система поддерживает три категории заявителей: физическое лицо, юридическое лицо, регулятор (ЦБ РФ) | P0 |
| FR-01.2 | Система поддерживает два типа обращений: письменное, устное | P0 |
| FR-01.3 | При регистрации фиксируются: номер обращения (автогенерация), дата регистрации, категория, тип, заявитель, ответственный | P0 |
| FR-01.4 | Каждому обращению назначается дедлайн (SLA) — дата и время ответа | P0 |
| FR-01.5 | Форма регистрации валидируется перед сохранением: обязательные поля — категория заявителя, тип, текст обращения | P0 |
