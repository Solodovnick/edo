## Черновик USM (по ролям)

### Ответственный специалист
| Активность | Story | Статус |
|---|---|---|
| Просмотр обращений | Вижу таблицу «Мои обращения» с фильтрами | ✅ MVP |
| Открытие карточки | Кликаю на строку → открывается карточка | ✅ MVP |
| Поиск клиента в CRM | Ввожу имя → автозаполнение данных | ✅ MVP |
| Подготовка решения | Вношу текст решения, сохраняю | ✅ MVP |
| Переназначение | Назначаю другого ответственного | P1 |
| Эскалация | Эскалирую обращение | P1 |

### Аудитор
| Активность | Story | Статус |
|---|---|---|
| Просмотр на аудит | Вижу обращения со статусом «Решено» | ✅ MVP |
| Проверка решения | Читаю исходное обращение и решение специалиста | ✅ MVP |
| Запись результата | Выбираю шаблон аудита, пишу комментарий | ✅ MVP |
| Публикация | Публикую результат аудита | ✅ MVP |

### Администратор
| Активность | Story | Статус |
|---|---|---|
| Управление пользователями | Создаю/редактирую учётные записи | P2 |
| Настройка SLA | Задаю сроки по категориям | P2 |
| Просмотр аналитики | Вижу общий Dashboard с KPI | P1 |
