# Creative & design

## When this skill applies

Use for tasks aligned with the **Creative & Design** category in [anthropics/skills](https://github.com/anthropics/skills/tree/main/skills).

## Instructions

1. Prefer existing design tokens, spacing scales, and components in the target codebase over ad-hoc values.
2. For pixel-level UI, reconcile layout with screenshots or design files; call out intentional deviations.
3. Keep copy and hierarchy readable; avoid decorative clutter unless the user asks for it.
4. If the repo has Code Connect, design rules, or Figma links, follow them before inventing new patterns.
